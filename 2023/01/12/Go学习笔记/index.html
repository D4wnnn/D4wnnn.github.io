Go学习笔记

Go学习笔记

简介

go无类和继承的概念 通过接口实现多态性

Docker是根据go开发的

go是编译型语言 自带编译器

在 Windows 下，标准库的位置在Go语言根目录下的子目录 pkg\windows_amd64 中；在 Linux 下，标准库在Go语言根目录下的子目录 pkg\linux_amd64 中（如果是安装的是 32 位，则在 linux_386 目录中）。一般情况下，标准包会存放在 $GOROOT/pkg/$GOOS_$GOARCH/ 目录下。

基本语法

  • 基本类型有哪些？

    • bool

    • string

    • int int8 int16、int32、int64

    • uint、uint8、uint16、uint32、uint64、uintptr

    • byte // uint8 的别名

    • rune // int32 的别名 代表一个 Unicode 码

    • float32、float64

    • complex64、complex128

      上述变量一旦声明就会全部自动初始化

  • 如何声明？

    var 变量名 变量类型

  • 如何批量定义变量？

  • 简短模式是什么？

    就是定义变量的同时必须显式初始化

    不能提供数据类型

    只能用在函数内部

    i,j := 1,2

  • 什么是“烫烫烫”和“屯屯屯”的梗？

    微软的 VC 编译器会将未初始化的栈空间以 16 进制的 0xCC 填充，而未初始化的堆空间使用 0xCD 填充，而 0xCCCC 和 0xCDCD 在中文的 GB2312 编码中刚好对应“烫”和“屯”字。

  • 注意：在多个短变量声明和赋值中，至少有一个新声明的变量出现在左值中，即便其他变量名可能是重复声明的，编译器也不会报错

  • 如何变量交换？

    a,b = b,a

  • 什么是匿名变量？

    就是一个下划线

    在Lua等编程语言中，匿名变量又称为哑元变量

  • 在go中变量的作用域是怎样的？

    • 局部变量

      函数内的局部变量可以和全局变量名称相同，但是会优先考虑局部变量

    • 全局变量

      全局变量必须以var关键词开头 如果想在外部包中使用，则首字母必须大写

    • 形式参数

      就是定义函数时函数名后面括号中的变量，形式参数会作为函数的局部变量来使用

  • 在go中很小或者很大的数最好用科学计数法书写

  • 在go中打印小数可以用”%.nf”控制有几位小数

  • 在go中布尔值不会隐式转换为数字0或者1

  • 在go中如何输入多行字符串？

    使用反引号

    在反引号中间的一切均不会被编译器识别 都是字符串的一部分

  • go的字符类型有两种：rune和uint8类型

    • uint8代表ascii的一个字符
    • rune代表一个utf-8字符
