使用最小花费爬楼梯

给你一个整数数组 cost ，其中 cost[i] 是从楼梯第 i 个台阶向上爬需要支付的费用。一旦你支付此费用，即可选择向上爬一个或者两个台阶。

你可以选择从下标为 0 或下标为 1 的台阶开始爬楼梯。

请你计算并返回达到楼梯顶部的最低花费。

分析：

首先dp[i]是爬到第i个楼梯的最小花费

初始化dp[0]=0,dp[1]=0

递推公式dp[i]=min{dp[i-1]+cost[i-1],dp[i-2]+cost[i-2]}

动态规划问题

这道题首先注意初始化问题，第一列和第一行都要初始化为1

dp[i][j]就是到第i行第j列共有多少不同的路径

因为是从上到下推，从左到右推，所以要初始化为1

注意在初始化时，range的左边界要是1

整数拆解

343. 整数拆分 - 力扣（Leetcode）

给定一个正整数 n ，将其拆分为 k 个 正整数 的和（ k >= 2 ），并使这些整数的乘积最大化。

返回 你可以获得的最大乘积 。

分析：

这道题也是dp问题，期中比较复杂的是搞清楚dp[i]中i的含义，dp[i]代表数字i的拆分数最大乘积，还要搞清楚递推关系，比如对于任意数字i，有两种方式进行拆解，一种是j(i-j)另一种是j*dp[i-j],取最大值,然后还要分清除递推的方向，是从前到后。外层循环是从前到后，内从循环是进行dp数组的更新
