232用栈实现队列-力扣-https://leetcode.cn/problems/implement-queue-using-stacks/description/

思路：

要想实现队列的先进先出，需要使用两个栈，一个入栈，一个出栈。对于push()操作，只需要简单的调用入栈的append()操作即可；对于pop()操作，首先判断两个栈是否都为空，然后判断出栈是否为空，最后将入栈的所有元素放到出栈里，执行出栈的pop()操作

六、回溯法

回溯法并不高效

回溯的本质是穷举，穷举所有可能，有些问题只能暴力解决

回溯法，一般可以解决如下几种问题：

  • 组合问题：N个数里面按一定规则找出k个数的集合
  • 切割问题：一个字符串按一定规则有几种切割方式
  • 子集问题：一个N个数的集合里有多少符合条件的子集
  • 排列问题：N个数按一定规则全排列，有几种排列方式
  • 棋盘问题：N皇后，解数独等等

所有回溯法的问题都可以抽象为树形结构

回溯三部曲：

  • 返回值及参数
  • 回溯函数终止条件
  • 回溯搜索的遍历过程

七、贪心算法

贪心的本质时选择每一阶段的局部最优，从而达到全局最优

那么什么时候用贪心呢？

贪心算法并没有固定的套路，如果局部最优不能推出来全局最优，就要使用动态规划

贪心算法分为下面四步：

  • 将问题分解为若干子问题
  • 找出贪心策略
  • 求每一个子问题的最优解
  • 将局部最优解堆叠成全局最优解

其实只要搞清楚局部最优是什么就好了
